# Voyager

Voyager is a library to control multiple servers and send remote commands.

Requiriments:


### Server Installation
```sh
$ ./voyager/houston.py 0.0.0.0 1333
```

### Client Installation
```sh
$ ./voyager/voyager1.py 0.0.0.0 1333
```

### Mission Control Installation
```sh
$ ./voyager/voyager2.py 0.0.0.0 1333
```
